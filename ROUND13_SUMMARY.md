### 1. ✅ เพิ่มข้อมูลธนาคารจาก Thunder API

**ปัญหา:**
- หน้าจัดการธนาคารว่างเปล่า
- ต้องการให้ใส่ข้อมูลธนาคารมาให้เลย

**วิธีแก้:**
- เพิ่มปุ่ม **"เพิ่มข้อมูลธนาคารจาก Thunder API"** ในหน้าจัดการธนาคาร
- คลิกปุ่มเดียวจะเพิ่มข้อมูลธนาคาร 18 แห่งอัตโนมัติ
- ข้อมูลจาก https://document.thunder.in.th/other/bank-codes
- ถ้าธนาคารมีอยู่แล้วจะอัพเดตข้อมูล (ไม่ลบ logo)

**วิธีใช้:**
1. เข้าหน้า **Admin > จัดการธนาคาร**
2. คลิกปุ่ม **"เพิ่มข้อมูลธนาคารจาก Thunder API"**
3. ยืนยันการเพิ่มข้อมูล
4. ระบบจะแสดงผลลัพธ์ (เพิ่มใหม่กี่ธนาคาร / อัพเดตกี่ธนาคาร)

### 4. ✅ แก้ไข Flex Message ไม่แสดงใน LINE

**ปัญหาหลัก:**
```
ModuleNotFoundError: No module named 'mongoengine'
```

**สาเหตุ:**
- ระบบใช้ MongoEngine ใน `models/bank.py`
- แต่ไม่ได้ติดตั้ง MongoEngine บน Heroku
- ทำให้ API error → Flex Message ไม่แสดง

**วิธีแก้:**
- **เปลี่ยนจาก MongoEngine เป็น PyMongo ทั้งระบบ**
- เขียน `BankModel` ใหม่ทั้งหมด
- แก้ไข API endpoints ทั้งหมดที่เกี่ยวข้อง

**ผลลัพธ์:**
- ✅ หน้าจัดการธนาคารทำงานได้ปกติ
- ✅ Flex Message แสดงได้พร้อม logo ธนาคาร
- ✅ สลิปซ้ำแสดงเป็น Flex Message พร้อมแถบเตือน

## 🚀 สิ่งที่ต้องทำหลัง Deploy

### ⚠️ สำคัญ: ต้องเพิ่มข้อมูลธนาคารก่อนใช้งาน

#### ขั้นตอนที่ 1: เพิ่มข้อมูลธนาคาร
1. เข้าหน้า **Admin > จัดการธนาคาร**
2. คลิกปุ่ม **"เพิ่มข้อมูลธนาคารจาก Thunder API"**
3. ยืนยันการเพิ่มข้อมูล
4. ระบบจะเพิ่มธนาคาร 18 แห่งอัตโนมัติ

#### ขั้นตอนที่ 2: อัพโหลด Logo ธนาคาร (ถ้าต้องการ)
1. คลิก **"แก้ไข"** ที่ธนาคารที่ต้องการ
2. อัพโหลดไฟล์ logo (PNG/JPG, แนะนำ 200x200px)
3. คลิก **"บันทึก"**
4. Logo จะแสดงใน Flex Message ทันที

**หมายเหตุ:** ถ้าไม่อัพโหลด logo ระบบจะใช้ logo default
