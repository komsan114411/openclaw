## การติดตั้งระบบ

### ความต้องการของระบบ
- Python 3.11 หรือสูงกว่า
- MongoDB (Local หรือ MongoDB Atlas)
- LINE Official Account
- OpenAI API Key (สำหรับ AI Chatbot)
- Thunder/SlipOK API Key (สำหรับตรวจสอบสลิป)

### ขั้นตอนการติดตั้ง

#### 1. แตกไฟล์ ZIP
```bash
unzip line-oa-improved.zip
cd line-oa-improved
```

#### 2. ติดตั้ง Dependencies
```bash
pip install -r requirements.txt
```

#### 3. ตั้งค่า Environment Variables
สร้างไฟล์ `.env` หรือแก้ไขไฟล์ที่มีอยู่:
```env
# MongoDB
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/
MONGODB_DATABASE=lineoa_system

# Application
SECRET_KEY=your-secret-key-here
DEBUG=False

# Optional: OpenAI (ถ้าต้องการใช้ AI)
OPENAI_API_KEY=sk-...

# Optional: Slip Verification
SLIP_API_PROVIDER=thunder
SLIP_API_KEY=your-slip-api-key
```

#### 4. รันโปรแกรม

**Development Mode:**
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

**Production Mode:**
```bash
gunicorn main:app -w 4 -k uvicorn.workers.UvicornWorker --bind 0.0.0.0:8000
```

#### 5. เข้าใช้งานระบบ
เปิดเบราว์เซอร์ไปที่: `http://localhost:8000`

## สำหรับ Admin

### Dashboard
หลังจากเข้าสู่ระบบ Admin จะเห็น:
- **สถิติภาพรวม:** จำนวนผู้ใช้, บัญชี LINE OA, บัญชีธนาคาร
- **ผู้ใช้ล่าสุด:** รายชื่อผู้ใช้ที่สร้างล่าสุด
- **บัญชี LINE OA:** รายการบัญชีทั้งหมดในระบบ

### การจัดการผู้ใช้

#### สร้างผู้ใช้ใหม่
1. ไปที่เมนู "จัดการผู้ใช้"
2. คลิก "เพิ่มผู้ใช้"
3. กรอกข้อมูล:
   - Username (ต้องไม่ซ้ำ)
   - Password (อย่างน้อย 8 ตัวอักษร)
   - Role: Admin หรือ User
   - ชื่อ-นามสกุล (ไม่บังคับ)
   - อีเมล (ไม่บังคับ)
4. คลิก "สร้างผู้ใช้"

#### ลบผู้ใช้
1. ไปที่เมนู "จัดการผู้ใช้"
2. หาผู้ใช้ที่ต้องการลบ
3. คลิกปุ่ม "ลบ"
4. ยืนยันการลบ

> ⚠️ **หมายเหตุ:** ไม่สามารถลบบัญชีตัวเองได้

### การจัดการบัญชี LINE OA

#### ดูบัญชี LINE OA ทั้งหมด
1. ไปที่เมนู "บัญชี LINE OA"
2. จะเห็นรายการบัญชีทั้งหมดในระบบ
3. แสดงข้อมูล:
   - ชื่อบัญชี
   - Channel ID
   - เจ้าของบัญชี
   - สถานะ
   - สถิติการใช้งาน

#### เพิ่มบัญชี LINE OA (สำหรับ User)
Admin สามารถเพิ่มบัญชี LINE OA ให้ User ได้โดย:
1. แจ้ง User ให้เข้าสู่ระบบ
2. User สามารถเพิ่มบัญชี LINE OA ของตัวเองได้

### การจัดการบัญชีธนาคาร

#### เพิ่มบัญชีธนาคาร
1. ไปที่เมนู "บัญชีธนาคาร"
2. คลิก "เพิ่มบัญชีธนาคาร"
3. กรอกข้อมูล:
   - ชื่อบัญชี (เช่น "บัญชีรับเงิน")
   - ธนาคาร (เลือกจาก dropdown)
   - เลขที่บัญชี
   - เชื่อมกับบัญชี LINE OA (ไม่บังคับ)
   - คำอธิบาย (ไม่บังคับ)
4. คลิก "บันทึก"

#### แก้ไขบัญชีธนาคาร
1. ไปที่เมนู "บัญชีธนาคาร"
2. คลิกปุ่ม "แก้ไข" ที่บัญชีที่ต้องการ
3. แก้ไขข้อมูล
4. คลิก "บันทึก"

#### ลบบัญชีธนาคาร
1. ไปที่เมนู "บัญชีธนาคาร"
2. คลิกปุ่ม "ลบ" ที่บัญชีที่ต้องการ
3. ยืนยันการลบ

> 💡 **เคล็ดลับ:** การลบเป็น Soft Delete ข้อมูลจะยังอยู่ในฐานข้อมูล

## สำหรับ User

### Dashboard
หลังจากเข้าสู่ระบบ User จะเห็น:
- **บัญชี LINE OA ของฉัน:** รายการบัญชีที่เป็นเจ้าของ
- **สถิติ:** จำนวนข้อความ, ผู้ใช้, สลิปที่ตรวจสอบ
- **ปุ่มเพิ่มบัญชี:** เพิ่มบัญชี LINE OA ใหม่

### การเพิ่มบัญชี LINE OA

#### ขั้นตอนที่ 1: เตรียมข้อมูลจาก LINE Developers
1. ไปที่ [LINE Developers Console](https://developers.line.biz/)
2. สร้าง Provider (ถ้ายังไม่มี)
3. สร้าง Messaging API Channel
4. คัดลอกข้อมูลต่อไปนี้:
   - **Channel ID**
   - **Channel Secret**
   - **Channel Access Token** (Long-lived)

#### ขั้นตอนที่ 2: เพิ่มบัญชีในระบบ
1. คลิกปุ่ม "เพิ่มบัญชี LINE OA"
2. กรอกข้อมูล:
   - ชื่อบัญชี (ชื่อที่ใช้แสดงในระบบ)
   - Channel ID
   - Channel Secret
   - Channel Access Token
   - คำอธิบาย (ไม่บังคับ)
3. คลิก "เพิ่มบัญชี"

#### ขั้นตอนที่ 3: ตั้งค่า Webhook
1. หลังจากเพิ่มบัญชีสำเร็จ ไปที่ "ตั้งค่า"
2. คัดลอก Webhook URL
3. กลับไปที่ LINE Developers Console
4. ไปที่ Messaging API > Webhook settings
5. วาง Webhook URL
6. เปิดใช้งาน Webhook
7. ปิด Auto-reply messages (ถ้าต้องการใช้ AI)

## การตั้งค่า LINE OA

### เข้าสู่หน้าตั้งค่า
1. ไปที่ Dashboard
2. คลิกที่บัญชี LINE OA ที่ต้องการตั้งค่า
3. คลิก "ตั้งค่า"

### การตั้งค่า AI Chatbot

#### เปิดใช้งาน AI
1. เปิดสวิตช์ "เปิดใช้งาน AI Chatbot"
2. กรอก **OpenAI API Key**
   - ไปที่ [OpenAI Platform](https://platform.openai.com/)
   - สร้าง API Key
   - คัดลอกมาวาง
3. เลือก **โมเดล AI:**
   - GPT-4.1 Mini (แนะนำ)
   - GPT-4.1 Nano (เร็วกว่า ถูกกว่า)
   - Gemini 2.5 Flash
4. กรอก **บุคลิกภาพ AI:**
   ```
   เป็นผู้ช่วยที่เป็นมิตรและช่วยเหลือดี 
   ตอบคำถามด้วยความสุภาพและให้ข้อมูลที่ถูกต้อง
   ```
5. คลิก "บันทึกการตั้งค่า"

### การตั้งค่าตรวจสอบสลิป

#### เปิดใช้งานตรวจสอบสลิป
1. เปิดสวิตช์ "เปิดใช้งานตรวจสอบสลิป"
2. เลือก **ผู้ให้บริการ API:**
   - Thunder API
   - SlipOK
3. กรอก **API Key** ของผู้ให้บริการ
4. เลือก **Template ตอบกลับ:**
   - คลิก dropdown "เลือก Template"
   - เลือก Template ที่ต้องการ
   - หรือคลิก "จัดการ Template" เพื่อสร้างใหม่
5. คลิก **"ทดสอบ API"** เพื่อตรวจสอบการเชื่อมต่อ
   - ถ้าสำเร็จจะแสดง "เชื่อมต่อสำเร็จ ✓"
   - ถ้าล้มเหลวจะแสดงข้อความผิดพลาด
6. คลิก "บันทึกการตั้งค่า"

## การตรวจสอบสลิป

### การทำงานของระบบ
1. ผู้ใช้ส่งรูปสลิปมาทาง LINE
2. ระบบตรวจจับว่าเป็นรูปภาพ
3. ส่งรูปไป API ตรวจสอบ
4. ได้ข้อมูลสลิป:
   - จำนวนเงิน
   - วันที่-เวลา
   - ธนาคาร
   - เลขบัญชี
   - หมายเลขอ้างอิง
5. ส่ง Flex Message กลับไปให้ผู้ใช้

### การจัดการ Template ตอบกลับ

#### สร้าง Template ใหม่
1. ไปที่ "จัดการ Template ตอบกลับสลิป"
2. คลิก "สร้าง Template ใหม่"
3. กรอกข้อมูล:
   - ชื่อ Template
   - คำอธิบาย
   - Template JSON (Flex Message)
4. คลิก "บันทึก"

#### ตั้งเป็น Template เริ่มต้น
1. ไปที่ "จัดการ Template ตอบกลับสลิป"
2. หา Template ที่ต้องการ
3. คลิก "ตั้งเป็นค่าเริ่มต้น"

#### ลบ Template
1. ไปที่ "จัดการ Template ตอบกลับสลิป"
2. หา Template ที่ต้องการลบ
3. คลิก "ลบ"
4. ยืนยันการลบ

> ⚠️ **หมายเหตุ:** ไม่สามารถลบ Template เริ่มต้นได้

1. ไปที่ https://platform.openai.com/
