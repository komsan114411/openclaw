🤖 Virtual Channels คืออะไร?

Virtual Channel คือช่องทางเสมือนที่ให้บอทภายนอก (เช่น Thunder Solution) เชื่อมต่อมาหาระบบของเราแทนที่จะเชื่อมตรง LINE

ประโยชน์: ควบคุมการสื่อสารทั้งหมด, ไม่ต้องแชร์ LINE credentials หลัก, ติดตามการใช้งานได้

🆕 สร้าง/Import Virtual Channel

สร้าง Virtual Channel ใหม่พร้อม credentials ที่เราสร้างให้

💡 สำหรับ Thunder Solution: ใช้ Channel ID และ Channel Secret จาก LINE ที่มีอยู่แล้ว

ระบบจะใช้ credentials เหล่านี้เพื่อเชื่อมต่อกับ LINE และสร้าง Access Token ใหม่สำหรับ API ของเรา

  2. ตั้งค่า Webhook URL เป็น: {{ base_url }}/virtual/{{ channel.channel_id }}/webhook
  3. ตั้งค่า API Base URL เป็น: {{ base_url }}/virtual-api/v2/bot
  4. ใช้ Access Token ในการเรียก API: {{ channel.access_token }}
  5. Thunder จะได้รับ events จาก LINE ผ่านระบบเราและสามารถส่งข้อความกลับได้
