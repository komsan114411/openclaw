คู่มือ Error Codes

รายละเอียดและคำอธิบาย Error Codes ทั้งหมดในระบบ LINE OA Management

Error Code คืออะไร?

Error Code คือรหัสที่ระบบใช้ระบุสาเหตุของข้อผิดพลาดหรือปัญหาที่เกิดขึ้น เมื่อระบบตอบกลับผู้ใช้ จะมีรหัส Error Code แสดงในวงเล็บท้ายข้อความ เช่น [404] เพื่อให้ผู้ดูแลระบบสามารถตรวจสอบและแก้ไขปัญหาได้อย่างรวดเร็ว

ประโยชน์ของ Error Code

  • ช่วยระบุสาเหตุของปัญหาได้อย่างรวดเร็ว
  • ทำให้การแก้ไขปัญหาเป็นระบบและมีมาตรฐาน
  • ผู้ใช้สามารถแจ้งปัญหาได้ชัดเจนด้วยรหัส Error
  • ผู้ดูแลระบบสามารถติดตามและวิเคราะห์ปัญหาได้

หมายเหตุ: คุณสามารถกำหนดข้อความตอบกลับสำหรับแต่ละ Error Code ได้ ผ่านหน้า ตั้งค่าขั้นสูง ของแต่ละบัญชี LINE OA

[404] ไม่พบข้อมูล
ไม่พบข้อมูล / Not Found
Error Code 404 เกิดขึ้นเมื่อระบบไม่สามารถค้นหาข้อมูลที่ต้องการได้ อาจเกิดจากการที่ API Key หมดอายุ แพ็คเกจหมดอายุ หรือข้อมูลไม่มีอยู่ในระบบ
สาเหตุที่เป็นไปได้:
  • API Key หมดอายุ (OpenAI, Slip Verification)
  • แพ็คเกจบริการหมดอายุ
  • ไม่พบข้อมูลในฐานข้อมูล
  • URL หรือ Endpoint ไม่ถูกต้อง
ตัวอย่างข้อความ:
ขออภัย ไม่พบข้อมูลที่ต้องการ กรุณาติดต่อผู้ดูแลระบบ [404]
วิธีแก้ไข:
  • ตรวจสอบและต่ออายุ API Key
  • ตรวจสอบแพ็คเกจบริการ
  • ตรวจสอบการตั้งค่าในระบบ
  • ติดต่อผู้ให้บริการ API
[405] เกินจำนวนที่กำหนด
เกินจำนวนที่กำหนด / Quota Exceeded
Error Code 405 เกิดขึ้นเมื่อการใช้งานเกินจำนวนที่กำหนดไว้ เช่น เกิน API Quota, Rate Limit หรือจำนวนข้อความต่อวัน
สาเหตุที่เป็นไปได้:
  • เกิน API Quota (OpenAI, Slip API)
  • เกิน Rate Limit (จำนวนครั้งต่อนาที/ชั่วโมง)
  • เกินจำนวนข้อความต่อวัน
  • เกินจำนวนผู้ใช้ที่อนุญาต
ตัวอย่างข้อความ:
ขออภัย คุณใช้งานเกินจำนวนที่กำหนด กรุณารอสักครู่หรือติดต่อผู้ดูแลระบบ [405]
วิธีแก้ไข:
  • รอให้ Quota รีเซ็ต (ตามเวลาที่กำหนด)
  • อัปเกรดแพ็คเกจบริการ
  • ลดความถี่ในการใช้งาน
  • ตรวจสอบและจำกัดจำนวนผู้ใช้
[409] ข้อมูลซ้ำซ้อน
ข้อมูลซ้ำซ้อน / Conflict
Error Code 409 เกิดขึ้นเมื่อมีข้อมูลซ้ำซ้อนในระบบ เช่น มีการทำรายการซ้ำ หรือข้อมูลที่ส่งมาขัดแย้งกับข้อมูลที่มีอยู่
สาเหตุที่เป็นไปได้:
  • มีข้อมูลซ้ำในฐานข้อมูล
  • มีการทำรายการซ้ำ
  • ข้อมูลขัดแย้งกัน
ตัวอย่างข้อความ:
ขออภัย พบข้อมูลซ้ำซ้อนในระบบ [409]
วิธีแก้ไข:
  • ตรวจสอบข้อมูลในระบบ
  • ลบข้อมูลซ้ำออก
  • ไม่ทำรายการซ้ำ
[500] ระบบขัดข้อง
ระบบขัดข้อง / Internal Server Error
Error Code 500 เกิดขึ้นเมื่อเกิดข้อผิดพลาดภายในระบบที่ไม่คาดคิด อาจเกิดจากบั๊กในโค้ด ฐานข้อมูลขัดข้อง หรือปัญหาอื่นๆ
สาเหตุที่เป็นไปได้:
  • ระบบขัดข้อง
  • ฐานข้อมูลไม่สามารถเชื่อมต่อได้
  • เกิดข้อผิดพลาดที่ไม่คาดคิด
  • บั๊กในโค้ด
ตัวอย่างข้อความ:
ขออภัย ระบบขัดข้อง กรุณาลองใหม่อีกครั้ง [500]
วิธีแก้ไข:
  • รีสตาร์ทระบบ
  • ตรวจสอบ logs เพื่อหาสาเหตุ
  • ตรวจสอบการเชื่อมต่อฐานข้อมูล
  • ติดต่อผู้ดูแลระบบ
[503] ระบบปิดปรับปรุง
ระบบปิดปรับปรุง / Service Unavailable
Error Code 503 เกิดขึ้นเมื่อระบบปิดปรับปรุงชั่วคราว เพื่อทำการบำรุงรักษาหรืออัปเดตระบบ
สาเหตุที่เป็นไปได้:
  • ระบบปิดปรับปรุง
  • บำรุงรักษาระบบ
  • อัปเดตระบบ
ตัวอย่างข้อความ:
ขออภัย ระบบปิดปรับปรุงชั่วคราว กรุณาลองใหม่ภายหลัง [503]
วิธีแก้ไข:
  • รอให้ระบบกลับมาใช้งานได้
  • ตรวจสอบประกาศจากผู้ดูแลระบบ
  • ลองใหม่ภายหลัง
