# Slip Template System Enhancement

## Task
ปรับปรุงระบบเทมเพลตสลิปให้มีฟีเจอร์ดังนี้:

1. **ชื่อผู้รับ/ผู้โอน**: แอดมินตั้งค่าได้จากหน้าเทมเพลตสลิป (ตอนสร้าง/แก้ไข)
2. **รวมฟีเจอร์**: รวม 'เทมเพลตสลิป' และ 'ตั้งค่าสลิป' เป็นหน้าเดียว
3. **รูปธนาคาร**: ดึงโลโก้ธนาคารมาแสดงในเทมเพลตเหมือนหน้าแอดมิน
4. **ปรับปรุง**: ให้ระบบทำงานร่วมกันได้ถูกต้อง + เพิ่ม validation

## Requirements
- สำรวจโค้ดเบสเอง หาไฟล์ที่เกี่ยวข้อง
- วิเคราะห์ database schema
- แก้ไขทั้ง backend (PHP/API) และ frontend (HTML/JS)
- ทดสอบให้ครบทุก case
- รองรับภาษาไทย
