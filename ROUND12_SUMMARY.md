### 3. ✅ แก้ไขการบันทึกการตั้งค่า AI

**ปัญหา:** 
- บันทึกการตั้งค่า AI แล้ว
- หลังรีเฟรชกลับมาเหมือนเดิม
- API Key หาย

**สาเหตุ:**
- JavaScript ส่ง placeholder (`********************************`) ไปบันทึกทับ API Key จริง

**การแก้ไข:**
- แก้ไข JavaScript ให้เช็คว่าเป็น placeholder หรือไม่
- ถ้าเป็น placeholder ก็ไม่ส่ง
- อัพเดท placeholder หลังบันทึกสำเร็จ

**ผลลัพธ์:**
- ✅ บันทึกการตั้งค่า AI ได้แล้ว
- ✅ API Key ไม่หายหลังรีเฟรช
- ✅ การตั้งค่าคงอยู่ถาวร

**วิธีใช้:**
1. เข้าหน้า **Settings > บอทตอบแชท (AI)**
2. กรอกข้อมูล:
   - เปิด/ปิด AI Chatbot
   - เลือก AI Model
   - กรอก OpenAI API Key (ถ้าต้องการเปลี่ยน)
   - กรอก System Prompt
   - ปรับ Temperature
   - กรอก Fallback Message
3. คลิก **บันทึกการตั้งค่า**
4. รีเฟรชหน้า → การตั้งค่ายังคงอยู่ ✅

### 4. ✅ แก้ไขชื่อบัญชีสีขาวมองไม่เห็น

**ปัญหา:** 
- ข้อความ "กำลังโหลด..." ในหน้าแชทสีขาวๆ มองไม่เห็น

**การแก้ไข:**
- เปลี่ยนสีจาก `gray-400` (อ่อน) เป็น `gray-700` (เข้ม)
- เพิ่ม `font-weight: 500` ให้ตัวอักษรหนาขึ้น

**ผลลัพธ์:**
- ✅ ข้อความมองเห็นชัดเจนแล้ว
- ✅ สีเข้มพอดี ไม่จ้าเกินไป

### 5. ✅ แก้ไข Logo บัญชีไลน์ไม่แสดงครบ

**สถานะ:**
- Logo ธนาคารแสดงได้แล้วหลังแก้ไข Flex Message (ข้อ 2)

**ถ้ายังไม่แสดง:**
- เป็นเพราะธนาคารนั้นยังไม่มี logo ใน database
- ต้องอัพโหลด logo เอง

**วิธีอัพโหลด Logo:**
1. เข้าหน้า **Admin > จัดการธนาคาร**
2. คลิก **แก้ไข** ที่ธนาคารที่ต้องการ
3. อัพโหลด logo (PNG/JPG, ขนาดแนะนำ 200x200px)
4. คลิก **บันทึก**
5. Logo จะแสดงใน Flex Message ทันที
